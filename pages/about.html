About FreshCart

FreshCart is a student-built grocery e-commerce project created to demonstrate core web development skills using HTML, CSS, and JavaScript. Our mission (in this simulated store) is to make fresh, high-quality groceries accessible and affordable for everyone.

This website is a university assignment and does not process real orders or payments. All data is stored locally in your browser using localStorage.

Our Values

  • Freshness first — quality produce, every time.
  • Fair prices for everyday shoppers.
  • Simple, friendly shopping experience.
